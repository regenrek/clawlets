# clawdlets-myproject

This repo is your fleet config (public-safe). Instance data lives in `.clawdlets/` (gitignored).

Quickstart
1) install `clawdlets` (global)
2) `clawdlets stack init`
3) `clawdlets secrets init`
4) `clawdlets doctor`
5) `clawdlets infra apply`
6) `clawdlets bootstrap`
7) join tailnet, then `clawdlets lockdown`

Docs: `docs/README.md`
